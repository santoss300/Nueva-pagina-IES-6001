📘 Descripción del Proyecto
Este proyecto consiste en el desarrollo del sitio web institucional del IES N° 6001, orientado a mejorar la comunicación académica y la presentación de la oferta educativa.
El sitio está construido,con un enfoque en el diseño adaptable (responsive design) y la accesibilidad.

La página principal incluye un carrusel dinámico de imágenes, secciones informativas, y un panel interactivo que presenta las distintas carreras disponibles en la institución.

🧠 Objetivos
Aplicar los conocimientos que aprenderemos en Programación de Aplicaciones para Dispositivos Moviles.
Implementar un sitio visualmente atractivo, funcional y accesible.
Representar fielmente la identidad institucional del IES N° 6001.
Utilizar buenas prácticas de desarrollo y estructura semántica del código.
